# Project Context

Projeto: wallet-service-api

Este projeto foi criado com objetivo de aprimorar e demonstrar arquitetura moderna para sistemas financeiros.

O projeto simula uma carteira digital (wallet).

---

## Comunicação
- apresentar respostas no idioma português.

## Funcionalidades principais

- criação de wallet
- ativação de conta
- gestão de saldo
- transferências entre wallets
- histórico de transações

---

## Objetivo técnico

O projeto também serve como laboratório para estudo de:

- arquitetura de microserviços
- mensageria
- observabilidade
- pipelines CI/CD
- cloud computing

---

## Evoluções planejadas

- integração com fila (RabbitMQ ou Kafka)
- processamento assíncrono
- uso de MongoDB para processamento de eventos
- deploy em ambiente cloud (AWS)

---

## Público alvo do projeto

- engenheiros de software
- arquitetos de sistemas
- profissionais estudando sistemas financeiros
